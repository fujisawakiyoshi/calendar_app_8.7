    デスクトップカレンダーアプリ README
─────────────────────────────

【アプリ概要】
----------------------------------------
本アプリは、予定の登録・編集・管理ができる
Windows用のデスクトップカレンダーアプリです。
ダークモード（愛称：かわいいモード）や祝日表示などにも対応しています。

【バージョン】
----------------------------------------
バージョン：v3.0.0

【動作環境】
----------------------------------------
・OS：Windows 11（動作確認済み）
・Python不要（exe実行のみで動作）
・必要なファイルは dist フォルダ内に同梱

【インストール・起動方法】
----------------------------------------
1. ダウンロードした「dist」フォルダを任意の場所に展開します。
2. distフォルダ内の「calendar_app.exe」をダブルクリックしてください。
　※初回起動時に多少時間がかかる場合があります。

【主な機能】
----------------------------------------
・カレンダー表示（月表示、日付クリックで予定一覧/追加/編集/削除）
・複数イベント登録、長文・記号入力対応
・祝日表示、自動で祝日セルに色付け
・ダークモード対応
・本日ボタン、月送り/月戻し、年跨ぎも対応
・ホバー効果（カーソルを予定の上に置くとツールチップで予定詳細を表示）
・キーボードナビゲーション（Deleteキー、Escキー操作）にも対応
・イベントデータはJSON形式で自動保存（dist/data/ 以下）

【ファイル構成例】
----------------------------------------
dist/
　├─ calendar_app.exe　… アプリ本体
　├─ data/
　│   ├─ events.json　… 予定データ（自動保存）
　│   └─ holidays.json … 祝日データ
　└─ その他同梱ファイル

【アンインストール方法】
----------------------------------------
1. アプリを終了してください。
2. アプリ本体（例：「calendar_app」フォルダ）をエクスプローラーで削除してください。
3. デスクトップ等に作成したショートカットも不要であれば削除してください。

※ 本アプリはWindowsレジストリやシステム設定を変更しません。フォルダ削除のみでアンインストールが完了します。

【トラブルシューティング】
----------------------------------------
● アプリが起動しない/画面が表示されない
　- ウイルス対策ソフトによりブロックされていないかご確認ください
　- Windows11で実行しているかご確認ください
　- distフォルダごと同じ場所に展開してください

● データが消えた・保存されない
　- dist/data/フォルダが削除・移動されていないかご確認ください
　- events.json, holidays.jsonが破損した場合は一度削除し再起動してください
　- バックアップは定期的にご自身で行ってください

● その他不明なエラー
　- test_result.log等のログをご参照ください
　- 改善しない場合は下記連絡先までご連絡ください
